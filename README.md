# axis_device

VirtualDeviceLib - Fiscal Device Management SDK for electronic receipt fiscalisation.

## Directory Structure

### `Library [DLL] [.Net 8.0] (Recommended) (Windows + Linux)/`
The main .NET 8.0 library distribution containing:
- `VirtualDeviceLib.dll` - Core fiscal device management library
- `VirtualTestTool.exe` / `V-LinuxTestTool.exe` - Test tools for Windows and Linux
- Runtime dependencies and native libraries for Windows, Linux, and macOS

### `Documentation/`
API documentation and reference materials:
- `VirtualDeviceLib DocumentationV2.3.0.pdf` - Latest API documentation
- `VirtualDeviceLib DocumentationV1.0.0.pdf` - Legacy API documentation
- `VirtualDevice Webservice Documentation.pdf` - Webservice reference
- `FDMS TEMPLATES REQUIREMENT (3).pdf` - FDMS template specifications
- `Virtual-Fiscalisation.png` - Fiscalisation diagram
- `Invoice - SubmitReceipt_Sample_Request.json` - Sample invoice request
- `CreditNote - SubmitReceipt_Sample_Request.json` - Sample credit note request
- `SampleResponses.json` - Sample API responses

### `Sample Code/`
Reference implementations:
- `VirtualTestTool.cs` - Windows sample code
- `V-LinuxTestTool.cs` - Linux sample code
